Minecraft 1.8.8-1.8.9 da calısan ve bircok sunucu lobisinde olan havai fişek eklentisi.

Özellikler
✅ Sağ Tıklama (Havaya Uçma): Oyuncu havai fişekle sağ tıkladığında, yukarıya doğru belirlenen bir yükseklikte zıplar.
✅ Sol Tıklama (İleri Uçma): Oyuncu havai fişekle sol tıkladığında, baktığı yöne doğru 2-3 blok ilerler.
✅ Slot Ayarlama: Oyuncu sunucuya girince eşya verilecegi slot.
✅ Cooldown Sistemi: Sol ve sağ tıklama için ayrı bekleme süresi (cooldown) bulunmaktadır.
✅ Config Desteği: Tüm ayarlar config.yml dosyasından özelleştirilebilir.



️ Kullanım
1️⃣ Oyuna girince oyuncuya otomatik olarak belirlenen slotta havai fişek verilir.
2️⃣ Sağ tıklayınca yukarı uçarsınız, sol tıklayınca ileri doğru hareket edersiniz.
3️⃣ Cooldown süresi dolmadan tekrar kullanamazsınız.
4️⃣ Adminler /pegasus reload komutuyla config.yml dosyasını yeniden yükleyebilir.
